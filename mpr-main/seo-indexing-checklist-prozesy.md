# Prozesy Branded SERP Fast-Track Checklist

Goal: Replace old branded results with Prozesy, Prozesy Media, and Prozesy Marketing as quickly as possible.

## 1) Google Search Console: Immediate Actions (Day 0)
1. Open Google Search Console and ensure both properties exist:
- Domain property: prozesy.com
- URL-prefix property: https://www.prozesy.com/
2. In Sitemaps, submit:
- https://www.prozesy.com/sitemap.xml
3. In URL Inspection, request indexing for priority URLs in this order:
- https://www.prozesy.com/
- https://www.prozesy.com/about/
- https://www.prozesy.com/contact/
- https://www.prozesy.com/services/
- https://www.prozesy.com/pricing/
- https://www.prozesy.com/google-ads/
- https://www.prozesy.com/meta-ads/
- https://www.prozesy.com/tiktok-ads/
- https://www.prozesy.com/seo-optimization/
- https://www.prozesy.com/blog/
4. Confirm crawlability for homepage:
- URL is on Google (or request indexing)
- Crawled page: successful
- Canonical selected by Google = https://www.prozesy.com/
5. In Settings -> Crawl stats, confirm Googlebot is fetching current pages.

## 2) If old domain still exists in Google
1. Keep old domain verified in Search Console.
2. If old domain is still active, use URL Inspection on old key URLs and confirm they redirect or canonically point to prozesy.com.
3. Use Removals only for urgent wrong snippets/URLs (temporary hide), not as primary migration strategy.

## 3) Entity Cleanup (Brand Signal Consolidation)
1. Google Business Profile:
- Business name should match target brand naming consistently.
- Website field must be https://www.prozesy.com/
- Services and business description should include Prozesy / Prozesy Media / Prozesy Marketing naturally.
2. Social profiles (LinkedIn, Facebook, Instagram, TikTok, Pinterest):
- Update display name/handle where possible to Prozesy branding.
- Update bio/about with the same brand wording.
- Website field on every profile = https://www.prozesy.com/
3. Third-party directories/portfolios:
- Update old brand mentions and URLs on major profiles and citations.
4. Contact consistency:
- Business name, phone, address, and primary website must match everywhere.

## 4) On-Site Brand Reinforcement
1. Homepage title and H1 should keep Prozesy prominently.
2. Homepage and About page should mention all brand variants naturally:
- Prozesy
- Prozesy Media
- Prozesy Marketing
3. Keep Organization/MarketingAgency schema and sameAs links up to date.
4. Keep robots.txt and sitemap.xml live and publicly accessible.

## 5) Weekly Follow-up (Day 3, 7, 14, 21)
1. Search Console Performance:
- Filter queries by: prozesy, prozesy media, prozesy marketing
- Track impressions/clicks and average position.
2. Search Console Indexing:
- Check indexed page count trend upward.
3. Re-request indexing for homepage/about/contact if snippets are still outdated.
4. Track branded SERP screenshots weekly to confirm transition.

## 6) Expected Timeline
- First movement: 3-10 days
- Branded SERP stabilization: 2-8 weeks
- Full entity replacement across all Google surfaces can take longer based on authority and citation cleanup.

## 7) Quick Execution Order (Best ROI)
1. Submit sitemap.xml
2. Request indexing for top 10 URLs
3. Update Google Business Profile website + brand wording
4. Update social profile names/bios/website links
5. Recheck branded queries in 3-7 days
